# Product Overview

Productivity Dashboard (Revo-U) is a lightweight, browser-based productivity tool designed to help users stay focused and organized. The application provides essential productivity features in a clean, distraction-free interface.

## Core Features

- Time-aware greeting display with real-time clock
- 25-minute Pomodoro focus timer
- Task list with CRUD operations
- Quick links manager for frequently accessed websites
- Light/dark theme toggle for user preference

## Target Use Case

Single-page dashboard for personal productivity, designed to be used as a browser start page or always-open tab. All data persists locally in the browser using Local Storage.

## Design Philosophy

- Minimal and focused: Only essential productivity features
- No backend required: Fully client-side application
- Privacy-first: All data stored locally in the user's browser
- Accessible and responsive: Works across devices and screen sizes
